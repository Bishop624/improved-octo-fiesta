# improved-octo-fiesta
The system integrates digital forensics techniques with machine learning to detect manipulation on images, documents and fake news content. It uses OCR for text extraction and metadata analysis for authenticity verification.
Fake News Detection Tools 
These tools analyze text, images, and sources to detect misinformation.
•	FactCheck.org – Verifies political and viral claims.
•	Snopes – One of the largest misinformation verification sites.
•	Google Fact Check Explorer – Searches fact-checked claims across the web.
•	Hoaxy – Visualizes how fake news spreads on social media.
Detection methods used
•	NLP (Natural Language Processing)
•	Source credibility analysis
•	Sentiment and claim verification
•	Social media propagation analysis

2. Detecting Font Inconsistencies in Documents 🔎
Used for identifying edited or forged documents like certificates or IDs.
Tools used
•	Adobe Photoshop – Used in forensic analysis to zoom and inspect pixel/font differences.
•	Forensically – Detects cloning, compression artifacts, and manipulation.
•	Amped Authenticate – Professional forensic tool for verifying image authenticity.
What investigators check
•	Font mismatch
•	Alignment differences
•	Kerning/spacing irregularities
•	Copy-paste artifacts
•	Pixel compression differences

3. Metadata Mismatch Detection 🧾
Metadata reveals when and how a file was created or edited.
Tools
•	ExifTool – Reads hidden metadata in images and documents.
•	FotoForensics – Detects manipulation through error level analysis.
•	Metadata2Go
Things investigators check
•	Creation date vs modification date
•	Device used to create the file
•	Editing software traces
•	GPS or location metadata

4. Fake ID or Forged Certificate Investigation 🪪
Common forensic verification methods include:
1. Document Examination

2. Digital Forensics
•	Metadata inspection
•	Error Level Analysis (ELA)

3. Database Verification
•	Checking serial numbers against official records
•	QR code verification
